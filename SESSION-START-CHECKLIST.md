# SESSION-START-CHECKLIST.md

## Every Session - Mandatory Read

### 1. Read Memory Files (IN ORDER)
- [ ] SOUL.md - Who I am
- [ ] USER.md - Who I'm helping  
- [ ] MEMORY.md - Long-term memory (main sessions only)
- [ ] memory/YYYY-MM-DD.md - Today's log
- [ ] memory/YYYY-MM-DD.md - Yesterday's log

### 2. Check System State
- [ ] Run `openclaw status`
- [ ] Check cron jobs: `openclaw cron list`
- [ ] Verify monitoring agent is running

### 3. Review Active Projects
- [ ] Check FILE-INDEX.md for current projects
- [ ] Review any TODO.md or active task files
- [ ] Check git status for uncommitted work

### 4. Ask User (If Unsure)
- [ ] "What are we working on today?"
- [ ] "Any context I should know?"
- [ ] "What time did we start?" (if relevant)

### 5. Never Assume
- [ ] Don't guess start times - ask
- [ ] Don't assume context - verify
- [ ] Don't rely on "mental notes" - write to files
